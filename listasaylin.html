LAS LISTAS EN HTML

Las listas en HTML nos permite crear conjuntos de elementos en forma de lista dentro de una página, todos los cuales irán precedidos, generalmente, por un guión o número.

LISTAS ORDENADAS

  2. carmen
  3. pedro
  4. maria
LISTAS ORDENADAS START
  1. julio
  2. carmen
  3. pedro
LISTAS ORDENADAS TYPE
  1. julio
  2. carmen
  3. daniel
LISTAS ORDENADAS REVERSED
  1. julio
  2. carmen
  3. pedro
